# Code of Conduct

## Expected Behavior

- Discuss technical issues directly and respectfully.
- Assume good intent, but challenge weak arguments with evidence.
- Keep review feedback about code and behavior, not identity.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language
- Deliberate disruption of collaboration spaces
- Publishing private information without permission

## Enforcement

Project maintainers may remove comments, reject contributions, or block participants who repeatedly violate these expectations.
